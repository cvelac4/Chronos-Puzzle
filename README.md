# Chronos-Puzzle
I have created a puzzle game with an infinite number of levels. There are three different difficulties (easy, medium, and hard) for the user to test out their puzzle solving skills. There is a leaderboard for users to see their scores. Each level is timed, and there is a move counter which tells the user how many moves her or she has used.  
This is the link to the game for anyone's interest. 

https://neon-chrono-quest.base44.app
